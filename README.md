Flight delays are a common problem in the aviation industry and affect both passengers and airlines in many ways. Delays can lead to missed connections, extra costs, and poor travel
experience. Many traditional systems used to handle delays depend on fixed rules and simple calculations, which are not always accurate. These methods often fail to consider how
different factors like weather, airline operations, and aircraft availability are connected. In this project, we use machine learning techniques in R to predict whether a flight will be
delayed or not. We worked with historical flight data and applied three models: Logistic Regression, Random Forest, and XGBoost. To improve the model performance, we created
additional features such as weather delay flag, carrier delay flag, and late aircraft flag. We also developed a Shiny-based application that allows users to enter flight details and get
instant predictions. This makes the system interactive and easy to use. After evaluating all models using metrics like accuracy, precision, recall, F1-score, and AUC, we observed that
XGBoost performed the best overall.This project shows that machine learning can be effectively used to predict flight delays and can be helpful in improving decision-making in the aviation sector.
